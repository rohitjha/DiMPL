Documentation

newtype Vector a

The Vector type is used to represent vectors. Internally, Vector is represented as a list.

For example:

>>> v1
<1,2>

>>> v2
<1,-1,1>

vDim :: Vector a -> Int

The vDim function returns the dimension or order of a Vector.

For example:

>>> vDim (Vector [1,1,3])

>>> vDim (Vector [1..27])

>>> vDim (Vector [])
